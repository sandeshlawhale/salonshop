# Professional Checkout Flow

A secure and professional B2B checkout experience for verified salon owners.

**DESIGN SYSTEM (REQUIRED):**
USE THIS THEME FOR ALL PAGES:
- Layout: Clean, airy, professional B2B luxury.
- Colors: White background, Neutral-900 text, Blue-600 accents (#0066CC).
- Border Radius: 24px-32px for cards and containers.
- Typography: Large bold headings, tight spacing.
- Elements: Use professional salon imagery, high-end product shots.
- Interaction: Soft hover states, shadow transitions.

**Page Structure:**
1. Progress stepper (Shipping -> Verification -> Payment).
2. Agent ID verification section with "Verified" badge.
3. Itemized order summary with B2B volume discount callouts.
4. Secure payment method selector (UPI, Net Banking, Card).
5. Trust footer with SSL and Verified Salon badges.
