# Design System: Salon E-Comm

## 1. Vision
A premium, professional B2B marketplace for salon owners. The aesthetic should be clean, luxury, and high-trust.

## 2. Color Palette
- **Primary**: `#0066CC` (Professional Blue)
- **Secondary**: `#4F46E5` (Indigo)
- **Neutral 900**: `#0A0A0A` (Deep Black for typography)
- **Neutral 500**: `#737373` (Medium Gray)
- **Background**: `#FFFFFF` (Pure White) / `#FAFAFA` (Light Gray)

## 3. Typography
- **Heading**: Inter/Manrope, ExtraBold, Tracking-tight
- **Body**: Inter, Medium/Regular

## 4. UI Patterns
- **Cards**: Large border radius (24px+), subtle border (`neutral-100`), soft shadow on hover.
- **Buttons**: Fully rounded (pill) or large radius (16px), bold text.
- **Icons**: Lucide-style, thin strokes (1.5px - 2px).

## 5. Design System Block for Stitch
```text
USE THIS THEME FOR ALL PAGES:
- Layout: Clean, airy, professional B2B luxury.
- Colors: White background, Neutral-900 text, Blue-600 accents (#0066CC).
- Border Radius: 24px-32px for cards and containers.
- Typography: Large bold headings, tight spacing.
- Elements: Use professional salon imagery, high-end product shots.
- Interaction: Soft hover states, shadow transitions.
```
